Как зарегистрироваться в системе Инфометр впервые?

Представителям организаций необходимо пройти регистрацию для того, чтобы иметь возможность повысить свои оценки, взаимодействуя с экспертами Фонда Свободы Информации.

Для регистрации в системе Инфометр вам необходимо:

1. Перейти по ссылке «Регистрация» в правом верхнем углу страницы либо перейти по данной ссылке {{ registration_url }}

2. Заполнить появившуюся регистрационную форму (звездочкой отмечены поля, обязательные к заполнению)

Если вы представляете организацию и хотите повлиять на выставленные сайту вашей организации оценки, в поле «Статус» вам необходимо выбрать «представитель организации».

Для поля «Пароль» рекомендуем создать сложный пароль, используя латинские буквы (A-Z, a-z) и цифры (0-9) для обеспечения безопасности.

В поле «Код приглашения» представитель организации вводит код, автоматически высланный в письме-приглашении на регистрацию.

После заполнения регистрационной формы нажмите на кнопку «Зарегистрироваться».

3. Подтвердить регистрацию.
После регистрации на указанный вами адрес электронной почты отправляется письмо. Для подтверждения регистрации необходимо перейти по ссылке внутри письма.

В каждом новом цикле мониторинга уже зарегистрированному представителю организации необходимо лишь ввести новый код приглашения в настойках профиля.

Уже зарегистрированным пользователям

Если вы уже зарегистрированы в Инфометре, войдите в систему, перейдите на страницу настроек по ссылке в правом верхнем углу, введите код в поле «Код приглашения». Теперь вы можете взаимодействовать с нашими экспертами: оставлять комментарии, повышать свои оценки и место организации в рейтинге.

Что такое код приглашения?

Код приглашения – это набор символов, которые необходимо ввести в соответствующее поле при заполнении формы регистрации для новых пользователей (см. вопрос № 1) либо в соответствующее поле раздела «Настройки» для пользователей, работавших ранее в системе (см. вопрос № 2). Код приглашения содержится в письме, которое в начале цикла мониторинга высылается на адрес электронной почты, указанный на сайте организации. Если по каким-либо причинам вы не получили код приглашения после начала цикла мониторинга, пожалуйста, свяжитесь с нашей службой технической поддержки по электронной почте tsupport@svobodainfo.org или тел. +7(812)766-03-66.



Что делать в системе Инфометр?

В Инфометре Вы можете:

  • ознакомиться с оценками, выставленными сайтам организаций;

  • повысить оценки сайта своей организации, следуя рекомендациям экспертов и внося изменения на свой сайт (доступно для зарегистрированных представителей организаций);

  • оставить свой комментарий к оценке по параметру или рекомендации эксперта. В комментарии вы можете выразить свое несогласие/сомнение относительно оценки либо оповестить нас о внесенных на сайт изменениях согласно рекомендациям экспертов (доступно для зарегистрированных представителей организаций);

  • заказать Сертификат информационной открытости вашей организации (документ, отражающий процент информационной наполненности сайта вашей организации по итогам цикла мониторинга).



Как оставить комментарий в Инфометре?

Данная функция доступна только для представителей организаций.

После успешной регистрации в системе (смотрите вопрос выше «Как зарегистрироваться в системе Инфометр?») представитель организации может оставить комментарий к оценкам своего органа власти в Инфометре.

Для этого вам необходимо:

1. Войти в систему Инфометр под своей учетной записью: перейти по ссылке «Войти» в правом верхнем углу.

2. Ввести адрес электронной почты и ваш пароль.

3. Перейти в раздел «Мои оценки» на панели быстрого доступа, расположенной слева, или в основном навигационном меню (слева вверху страницы).
Также вы можете увидеть оценки своего органа власти, кликнув по названию цикла мониторинга текущего года на главной странице Инфометра.

В разделе «Мои оценки» содержатся оценки наших экспертов по всем параметрам мониторинга для вашего органа.


4. Кликнуть на интересующий вас параметр.

5. Ввести текст комментария в поле ввода с заголовком «Ваш комментарий».

После того, как комментарий оставлен, вы увидите сообщение о сроке ответа на него. Время, отводимое эксперту для ответа на ваш комментарий, зависит от цикла мониторинга, и, как правило, не превышает 3 рабочих дней. Об этих действиях может направляться уведомление на указанный вами при регистрации адрес электронной почты, если вы активизировали данную функцию в разделе «Настройки».

Как посмотреть рейтинг организаций?

Для этого необходимо перейти на страницу «Рейтинги» по ссылке в главном меню (сверху слева), кликнуть на интересующий цикл мониторинга из списка.

Вы можете выстраивать рейтинг в зависимости от вида параметров: по всем параметрам, по параметрам на основе НПА, по рекомендательным параметрам, по параметрам, выбранным вами (пользовательский рейтинг).

Как восстановить пароль?

Если вы забыли пароль, его можно восстановить, перейдя по ссылке «Войти» справа на главной панели управления, а затем по ссылке «Забыли пароль?».

Введите в поле «Электронная почта» адрес электронной почты, который вы указывали при регистрации в системе Инфометр. На этот адрес будет отправлено письмо со ссылкой, перейдя по которой, можно будет ввести новый пароль.

Форма обратной связи

В подвале (поле внизу страницы) Инфометра расположена гиперссылка на форму обратной связи для обращений, вопросов, замечаний.
